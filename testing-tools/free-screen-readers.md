# VoiceOver (OSX)

## Installation

None! Just press <kbd>CMD</kbd> + <kbd>F5</kbd>.

## Usage

Keyboard shortcuts: https://www.apple.com/voiceover/info/guide/_1131.html

(Note that 'VO' is <kbd>CTRL</kbd> + <kbd>ALT/OPTION</kbd>)
